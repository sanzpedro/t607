### Commands

* cd (change directory)
* pwd (print working directory)
* ls (list all files cwd)
* mkdir (make directory)
* touch **<file_name>**

### GIT

* git init
* git remote add origin <repo_url>
* git status
* git add <file_1> <file_2> [a]
* git commit -m <"msg">
* git push origin <branch_name>
* git branch <new_branch>
* git checkout <branch_name>
